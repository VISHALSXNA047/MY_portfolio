# 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, education, and contact information through a clean, interactive, and user-friendly interface.

## 🚀 Live Demo

🔗 https://vishalsxna047.github.io/MY_portfolio/

## 📌 Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and visually appealing user interface
- Smooth scrolling navigation
- About Me section
- Skills showcase
- Projects portfolio section
- Contact section
- Social media integration
- Interactive elements powered by JavaScript
- Clean and organized layout for better user experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome

## 🎯 Purpose

The purpose of this project is to create a professional online presence and provide a platform to showcase my technical skills, projects, educational background, and contact information. The website allows visitors and recruiters to explore my work and learn more about my experience as a web developer.

## ✨ Key Highlights

- Built entirely using core web technologies without any frameworks.
- Designed a fully responsive layout that adapts seamlessly across different devices.
- Implemented interactive UI elements using JavaScript for enhanced user engagement.
- Structured content to effectively showcase projects, skills, and personal information.
- Focused on creating a clean, intuitive, and accessible user experience.
- Demonstrates front-end development, responsive design, and UI/UX fundamentals.
